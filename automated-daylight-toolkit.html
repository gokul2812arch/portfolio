Complete automation of daylight simulation - from geometry generation to report generation
Features of this tool :
Automated Geometry Generation from just Polylines
Interactive dashboard for Intuitive layman usage
Most accurate and automated sky file generation for simulation
Instant result visualization integrated with geometry
Automated Report Generation for LEED v4 compliance
Compliance Modelling 5x Faster
This tool has been developed to augment the efficiency and speed of the daylight compliance Modeling workflow and inculcate automation to abate human errors in manual processes. The expedited process with this tool is calculated to be 5-6 times faster than the usual workflow.
